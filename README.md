# 简易天猫商城
             开发工具：Myeclipse 10
             运行环境：apache-tomcat-9.0.19
             开发技术：jsp+servlet
             数据库：mysql 5.7
# 功能模块
             后台管理系统
             购物车模块
             订单模块
             用户注册模块
             用户登陆模块

# 版本
             后续将用Spring+Struts+Hibernate框架更新版本。
